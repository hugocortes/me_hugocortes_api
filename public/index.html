Hugo's Raspberry Pi API Server

Read Sensor Data From Arduino

  • URL
    • /radio/f:arduino_id/sensor/:sensor_type
  • Method: GET
  • URL Params
    • Required:
      • arduino_id=[integer]
      • sensor_type=[string]
        • temp
        • heat
        • humidity
  • Data Params:
    • None
  • Success Response:
    • Code: 200
    • Content: {"status":"success","id":"f1","sensor":"temp","sensor_val":"75.739998"}
  • Error Response:
    • Todo
  • Sample Call: http://api.hugocortes.me/radio/f1/sensor/temp
